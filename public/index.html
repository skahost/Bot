Commands

Slash Commands

All commands use Discord's native slash command system — type / to discover them.

/manage User
Interactive server control panel with Start / Stop / Restart / Kill / Reinstall / Stats buttons.
/help User
Shows all available commands split into Admin and User sections.
/status Admin
Live panel health dashboard — server count, node count, and overall status.
/list-servers Admin
Paginated list of all servers on the panel with live resource usage.
/server-info Admin
Detailed info for a specific server — owner, node, limits, and current stats.
/nodes Admin
Overview of all nodes with allocated vs. total RAM, CPU, and disk.
/create-server Admin
Create a new server on the panel with customizable resource limits.
/delete-server Admin
Permanently delete a server from the panel. Irreversible — use with caution.
/suspend-server Admin
Suspend a server, blocking the owner from starting or accessing it.
/unsuspend-server Admin
Lift a suspension and restore server access for the owner.
/bypass Admin
Enable or disable the 24-hour auto-stop exemption for a specific server.
/ping Admin
Bot latency, Discord API latency, and panel reachability check.
